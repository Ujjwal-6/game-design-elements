game design elements :

trex game 

characters -  characters are a set of objects or players in a game which have a particular role in the game , it can be a non playing character or a playing character : trex - PC , obstacles,clouds are NPC . 

Story : story is very essential for a game like it defines the main agenda of playing the game , using a story in a game can be very attractive and fun to play . in trex the story is that a dinousaur is running to escape from a city where he finds many obstacles in his path and if he hits any obstacles he dies.

Goals : goal in a game is something to achieve in a particular game . in trex , this a infinite game so there is no definite goal but in certain games which is not infinite can have a goal.

Rules: rules in a game are a set of orders which have to be followed in a game , in trex there are rules where u cannot hit the obstacle and if you hit you will die , you cannot jump before thee trex touches the ground , 

Balance : it where the game should not be that tough nor that much easy .

adaptivity : how the game adapts to different situations . in trex as your score increases the speed of your games increases

chance vs skill : this where u get particular lives and the skill to play a game . in trex there are no lives but there is a skill required to play such infinite games 

feedback : in game when you lose or win there have to something which is displayed on the screen , in trex there is score and high score displayed and when you lose there is game over displayed and a button to restart the game is also displayed
